Morgridge Software Assurance

Do It Early, Do It Often

Morgridge Software Assurance is a service that provides continuous software assurance capabilities to developers and researchers.

This no-cost code analysis service is open to the public. Let us help you to build better, safer, and more secure code today!

Rather than spending time installing, licensing and configuring software assessment tools on your own machine, let us do the work for you.

If you are an educator, open source developer or other non-profit software developer, then we can help you to analyse and improve your software code. To get started writing better code, contact us to set up a web based instance of the software assurance platform, and then you can upload and analyse your code in a way that is private and secure.

Get results in just four steps

1

 Create an instance

Contact us to have an instance of the web based tool created for you.

2

 Upload a package

Upload your code to the application. Rest assured that it will remain secure.

3

 Run an assessment

Create and run an assessment by choosing a package, tool, and platform.

4

 View your results

View assessment results. Sort and filter by weakness type or file.
